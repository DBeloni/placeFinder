# 📍 PlaceFinder

O **PlaceFinder** é uma aplicação web interativa projetada para mapear, gerenciar e localizar pontos de interesse em tempo real. Com uma interface limpa, moderna e responsiva, o sistema oferece um fluxo seguro de ponta a ponta, desde o controle de acesso até a exibição dinâmica do mapa.

---

## 🚀 Funcionalidades Principais

* **Autenticação Real com Firebase:** Sistema completo de Cadastro (`Sign Up`) e Login (`Sign In`) integrado ao Firebase Auth, com tratamento de erros em português.
* **Guarda de Rotas (Security Layer):** Proteção rigorosa de páginas privadas via `localStorage` e validação de estado, blindando o mapa contra acessos não autorizados via URL direta.
* **Web Components Nativos:** Header, Footer e elementos comuns modulares e reutilizáveis, garantindo performance e isolamento de escopo sem dependências pesadas.
* **Identidade Visual Consistente:** Design moderno, paleta de cores otimizada para legibilidade e Favicon vetorial (SVG) integrado para uma experiência de produto premium.
* **Histórico de Git Organizado:** Projeto estruturado seguindo o fluxo de ramificação Git, mantendo a branch `main` sempre estável e pronta para produção.

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3 (Variáveis nativas, CSS Grid e Flexbox), JavaScript Moderno (ES6+ Modules).
* **Backend & Security:** Firebase Authentication.
* **Componentização:** Custom Elements (Web Components).
